Add hover informations with Tags

You can add visual tags on your leadsheet.
You just need to create an array containing 'startbeat' and 'endbeat' value.
You can optionnally add a name and a color.
Here is an example 

{'startBeat': 1, 'endBeat': 5, 'name': 'First bar', 'color': "#559"}
